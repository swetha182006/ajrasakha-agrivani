# AgriVani
**A Voice-First AI Eligibility Engine for Indian Agricultural Schemes**

## Project Overview
AgriVani is an intelligent, voice-activated bridge designed to simplify the complex landscape of Indian government agricultural schemes. Built for the **Ajrasakha Hackathon Season 2**, this project addresses **Problem Statement 5: "Niti-Setu"**.

By extracting expert-validated rules directly from official government PDFs and utilizing a Python-based RAG (Retrieval-Augmented Generation) engine, AgriVani allows farmers to check their eligibility for schemes through simple voice queries in regional languages (English & Hindi). 

## Key Features (Phase 1 Target)
* **Voice-to-Text Input:** Accessible UI for low digital literacy.
* **Document-Backed AI:** Decisions based strictly on official PDFs, preventing AI hallucinations.
* **Cost-Effective Architecture:** Utilizing semantic caching to minimize API calls.
* **MERN Integration:** Designed to plug seamlessly into the Ajrasakha ecosystem.

## Tech Stack
* **Frontend:** React.js + TypeScript
* **Backend:** Node.js + Express.js + TypeScript
* **AI/Engine:** Python + FastAPI + LangChain (RAG)
* **Database:** MongoDB Atlas
